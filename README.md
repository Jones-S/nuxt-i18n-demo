## Setup

- Use correct node version via [Node Version Manager](https://github.com/nvm-sh/nvm):

```bash
nvm use
```

- Make sure to install the dependencies:

```bash
pnpm install
```

Start the development server on `http://localhost:3000`

```bash
pnpm dev
```
